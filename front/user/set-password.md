# 设置我的密码

## URI

```
/set-password
```

## 结构

| key | 说明 | 备注 |
| --- | --- | --- |
| error | 错误提示 |  |
| password | 密码 |  |
| password_error | 密码错误提示 |  |
| password_autofocus | 光标聚焦在邮箱密码 | autofocus 或者 不传 |
| confirm_password | 密码校验 |  |
| confirm_password_error | 密码校验错误提示 |  |
| confirm_password_autofocus | 光标聚焦在邮箱密码校验 | autofocus 或者 不传 |
