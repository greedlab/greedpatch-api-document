# 工程添加成员

## URI

```
/projects/:id/member
```

## 请求方式

```
DELETE
```

## 包头

```
Authorization: Bearer <token>
```

## 请求参数

| key | 类型 | 是否必须 | 说明 | 备注 | 例子 |
| --- | --- | --- | --- | --- | --- |
| :id | string | 是 | 工程 ID |  |  |
| email | string | 是 | 邮箱 |  |  |

## 成功

### 成功返回状态码

| 状态码 | 说明 | 备注 |
| --- | --- | --- |
| 200 | 删除成功 | 不管有没有对应的 email 都返回成功 |

### 成功返回数据实例

```json
{
  "message": "删除成功"
}
```

## 失败

### 失败返回状态码

| 状态码 | 说明 | 备注 |
| --- | --- | --- |
| 401 | token 失效 |  |
| 403 | 无权限访问工程/无权限删除成员 |  |
| 422 | 工程不存在 |  |
| 500 | 服务器内部错误 |  |

### 出错返回数据实例

```json
{
  "message": "工程不存在"
}
```
