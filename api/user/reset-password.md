# 重置密码

发送重置密码链接到邮箱。后端生成新的 token，重置密码链接为 `<front >/user/set-password?token=<token>`

## URI

```
/user/reset-password
```

## 请求方式

```
post
```

## 请求参数

| key | 类型 | 是否必须 | 说明 | 备注 | 例子 |
| --- | --- | --- | --- | --- | --- |
| email | string | 是 | 邮箱 |  |  |

## 成功返回数据

| key | 类型 | 说明 | 备注 |
| --- | --- | --- | --- |
| message | string | 提示信息 |  |

### 成功返回数据实例

```json
{
  "message": "请通过邮件里的链接重新设置密码"
}
```

## 出错返回数据

| 状态码 | 状态信息 | 说明 | 备注 |
| --- | --- | --- | --- |
| 422 | 邮箱格式错误/重置密码邮件发送失败 |  |  |
| 500 | 服务器内部错误 |  |  |

### 出错返回数据实例

```json
{
  "message": "邮箱格式错误"
}
```
