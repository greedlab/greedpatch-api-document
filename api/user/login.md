# 登录

## URI

```
/user/login
```

## 请求方式

```
post
```

## 请求参数

| key | 类型 | 是否必须 | 说明 | 备注 | 例子 |
| --- | --- | --- | --- | --- | --- |
| email | string | 是 | 邮箱 |  |  |
| password | string | 是 |  密码 | 未加密 |  |

## 成功返回数据

| key | 类型 | 说明 | 备注 |
| --- | --- | --- | --- |
| token | string | 邮箱 |  |
| role | int | 角色 | [详细说明](/table/user.md#role) |

### 成功返回数据实例

```json
{
  "token": "IFEI.rewqri.423jkdsf",
  "role" : 0
}
```

## 出错返回数据

| 状态码 | 状态信息 | 说明 | 备注 |
| --- | --- | --- | --- |
| 401 | 用户名或密码错误 |  |  |
| 500 | 服务器内部错误 |  |  |

### 出错返回数据实例

```json
{
  "message": "用户名或密码错误"
}
```
