# 删除补丁

管理员或工程管理员可操作

## URI

```
/patches/:id
```

## 请求方式

```
DELETE
```

## 包头

```
Authorization: Bearer <token>
```

## 请求参数

| key | 类型 | 是否必须 | 说明 | 备注 | 例子 |
| --- | --- | --- | --- | --- | --- |
| :id | string | 是 | 补丁 ID |  |  |

## 成功

### 成功返回状态码

| 状态码 | 说明 | 备注 |
| --- | --- | --- |
| 204 | 删除成功 | |

## 失败

### 失败返回状态码

| 状态码 | 说明 | 备注 |
| --- | --- | --- |
| 401 | token 失效 |  |
| 403 | 无权限 |  |
| 422 | 删除失败 |  |
| 500 | 服务器内部错误 |  |

### 出错返回数据实例

```json
{
  "message": "删除失败"
}
```
